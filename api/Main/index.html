Main

Provides Luau compilation and execution methods.

Internally uses LuauCeption for compiling code and Fiu for the Luau VM.

Calling the module itself is an alias for vLuau.execute.

Types​

CompilationOptions​

</>
type CompilationOptions = {
optLevel?: number,
debugLevel?: number,
typeInfoLevel?: number,
coverageLevel?: number
}

Options for compiling Luau source code.

Default compilation options from LuauCeption:

Name Default Value
optLevel 1
debugLevel 1
typeInfoLevel 0
coverageLevel 0

Main.execute(
source: buffer | string,--

The Luau source code or bytecode to execute.

chunkname: string?,--

The name of the chunk (for error messages). Defaults to (main) if not provided.

The environment to execute the code in. Defaults to the caller's environment.

Options for compiling the source code (if source is a string).

Options for executing the bytecode. You will likely not need to use this.

) → ...any--

The result of executing the chunk. Unless it has an explicit return, this will be nil.

Executes Luau bytecode or source code.

If the source is a string, it will be compiled first using the compilation options, if provided.
